# Support

Use GitHub Issues for support:

- Bug reports: https://github.com/astandrik/setup-local-ydb/issues/new?template=bug_report.yml
- Feature requests: https://github.com/astandrik/setup-local-ydb/issues/new?template=feature_request.yml
- General questions: https://github.com/astandrik/setup-local-ydb/issues

When reporting a problem, include:

- workflow YAML snippet;
- action version, for example `astandrik/setup-local-ydb@v1`;
- `version`, `tenant`, and `auth` inputs;
- relevant action log excerpt with secrets removed;
- expected and actual behavior.

Do not post passwords, tokens, private repository names, or sensitive runner logs.
